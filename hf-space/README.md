# BTCFi API Explorer

Interactive demo for the [BTCFi API](https://btcfi.aiindigo.com) — a Bitcoin Intelligence API providing 33 endpoints across Bitcoin, Ethereum, and Solana.

## Features
- Query real-time Bitcoin fees, mempool data, and address info
- Access AI-powered intelligence: whale detection, risk scoring, fee predictions
- Cross-chain support: Ethereum gas, SOL fees, and more
- No API key required — 100 free calls/day per IP
